Who is RBEH?

RBEH is the online identity of Ansari Mohammad Ahesan Ahmed (Jafar), a Diploma in Information Technology student with a strong interest in cyber security, web development, and modern technologies. RBEH Multiverse was created with the vision of making cyber security education more practical, interactive, and accessible for everyone. By combining learning resources, security tools, and realistic cyber demonstrations into one platform, the project encourages ethical learning, continuous skill development, and responsible use of technology.

📜

Internship Project Declaration

RBEH Multiverse was developed as part of the 15-Day Cyber Security Internship under the GTU | IBM SkillsBuild Skill-Based Training Program 2026. Throughout this internship, the project was designed and developed by applying practical cyber security concepts, secure web development practices, and modern front-end technologies. The objective of this project is to provide an interactive learning platform that combines cyber security tools, educational courses, and safe demonstration labs into a single professional web application while enhancing practical knowledge and real-world learning experience.
